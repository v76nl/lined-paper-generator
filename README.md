# Lined Paper Generator

Webブラウザ上でシミュレーションしながら、印刷用の罫線メモ用紙（A4サイズ）をカスタマイズしてPDF出力できるWebアプリケーション。

## 概要

ReactとTypeScriptを用いて構築したツール。
GUIからページ数、行数、罫線の太さや種類、色、行番号の有無などを直感的に調整し、ブラウザの印刷機能（@media print）を利用して正確なA4サイズのPDFとして書き出すことが可能。

## 技術スタック

- **Framework**: React 19
- **Build Tool**: Vite
- **Language**: TypeScript
- **Package Manager**: pnpm
